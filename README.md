## 簡介


**book review** 是一個使用[next.js](https://nextjs.org/) 和 [react](https://reactjs.org/) 開發的書籍收藏網站。
提供使用者搜尋、評分評論、收藏書籍，書本資料來自 google book API。

## 現有功能

- 註冊登入帳號
- 關鍵字搜尋書籍
- 對書籍評論、評分、加入書櫃
- 書櫃書籍閱讀狀態瀏覽、編輯

其餘功能持續追加中
